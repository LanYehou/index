HTML基本格式注释

<!DOCTYPE html> #请始终向 HTML 文档添加 <!DOCTYPE> 声明，这样浏览器才能获知文档类型。
<html>

<head> #头

<meta charset="UTF-8"> #指定编码类型，没有会乱码
<title>...</title> #标题

</head>

<body> #主体

<h1>...</h1> #段落标题
<p>...</p> #段落
<h2>...</h2>
<p>...</p>

<p>...<u>...</u></p> #<u>...</u>中有下划线

<div class:...> #分区或节，class定义类
<h6>...</h6>
<p>...</p>
</div>

<a href="http://...">...</a> #超链接

<ol> #有序列表
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>

<ul> #无序列表
<li>...</li>
<li>...</li>
<li>...</li>
</ul>

<img src="..." alt="..." /> #插入图片

<video src="..." controls="controls"> #插入视频，controls：显示控件
...
</video>

</body>
</html>
