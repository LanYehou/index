Shadowsocks 的部署

#服务器系统Liunx CentOS，需要Python 和pip

-----------------

pip install shadowsocks #安装

pip install shadowsocks #安装
ssserver #查看命令行
vim /etc/config.json #新建一个config文件

------------------

#config.json的注释

{
"server":"0.0.0.0", #服务的IP地址（0.0.0.0即为绑定所有IP，一般不需要改）
"server_port":****, #服务端口
"local_address":"127.0.0.1", #本地地址（不用改）
"password":"******", #密码
"timeout":300, #链接超时/s
"method":"aes-256-cfb", #加密方式
"fast_open":false, #false为关，true为开
"workers":1 #进程数
}

#服务启动

------------------
ssserver -c /etc/config.json -d start #从config.json载入信息并开始服务，start改为stop停止。

------------------
我的模板:

{
"server":"0.0.0.0",
"server_port":8388,
"local_address":"127.0.0.1",
"password":"l123456/",
"timeout":300,
"method":"aes-256-cfb",
"fast_open":false,
"workers":1
}
------------------
启动模板

ssserver -c /etc/ss.json -d start

------------------
快速启动：
ssserver -p 8388 -k l123456/ -m aes-256-cfb
